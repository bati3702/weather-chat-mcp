# 🌦️ Weather Israel MCP Server (Playwright)

פרויקט זה מציע שרת MCP (Model Context Protocol) מותאם אישית המאפשר לכל מודל שפה (LLM) לשלוט בדפדפן באופן אוטומטי לחלוטין ולשלוף מידע בזמן אמת על מזג האוויר בישראל מאתר [Weather2day](https://www.weather2day.co.il/forecast).

הפרויקט עושה שימוש ב-**Playwright** לצורך ניווט ואוטומציה חכמה בדפדפן, ובמנגנון **RAG (Retrieval-Augmented Generation)** כדי להעביר את המידע ישירות לתוך שיחת הצ'אט.

---

## 🎯 מטרת הפרויקט
* פיתוח שרת MCP מותאם אישית לחשיפת פונקציות (Tools) ל-LLM.
* ביצוע אוטומציה מלאה ואינטראקציה חכמה עם דפדפן (ללא מגע יד אדם).
* פתרון בעיות סנכרון ו-Flakiness בדפדפן באמצעות המתנה אקטיבית לשינויי תוכן בדף.
* שליפת התוכן של עמוד התחזית והזרקתו בחזרה לקונטקסט של המודל כדי לאפשר מענה מקיף ומעובד בתוך הצ'אט.

---

## 🛠️ טכנולוגיות בהן נעשה שימוש
* **Python**
* **FastMCP (MCP SDK)** - הספרייה הרשמית של Anthropic להגדרת פרוטוקול MCP.
* **Playwright (Python Async API)** - ספריית אוטומציה ושליטה בדפדפן Chromium.
